Introduction

In the previous papers of this series we asked “What is 5858?” and answered it on four levels: as a numeral (Paper 01), as a von Neumann ordinal (Paper 02), as the value of a unique morphism out of a Natural Number Object (Paper 03), and as the representable presheaf Hom(−,58)\mathrm{Hom}(-, 58) (Paper 04). Each refinement traded ontological substance for relational structure. The question that survives Paper 04 is whether the categorical viewpoint is itself an encoding – whether “58 as the representable Hom(−,58)\mathrm{Hom}(-,58)” replaces one substantialist commitment with another.

Homotopy type theory answers “no” constructively. In HoTT every type is already identified up to equivalence by axiom, so the question “which set, presheaf, or initial object is the real ℕ\mathbb{N}?” is dissolved into the slogan

equivalent things are equal.

Two natural number objects, two presentations of the reals, two K(G,1)K(G,1)’s for the same group are not merely isomorphic up to a chosen comparison map – they are paths in the universe.

This paper develops the perspective in detail. We restrict ourselves to the Book HoTT of  except where noted, and present the material in standard Martin-Löf style with one universe 𝒰\mathcal{U} for clarity; the ramified hierarchy 𝒰0:𝒰1:𝒰2:⋯\mathcal{U}_0 : \mathcal{U}_1 : \mathcal{U}_2 : \cdots is needed for foundational coherence but plays no role in the arithmetic content.

Notational conventions

We write a:Aa : A for “aa is a term of type AA,” f:A→Bf : A \to B for the type of (non-dependent) functions, ∏x:AB(x)\prod_{x:A} B(x) for the dependent product (“Π\Pi-type”), and ∑x:AB(x)\sum_{x:A} B(x) for the dependent sum (“Σ\Sigma-type”). The identity type of a,b:Aa, b : A is written a=Aba =_A b or 𝖨𝖽A(a,b)\mathsf{Id}_A(a, b). The constant path is 𝗋𝖾𝖿𝗅a:a=Aa\mathsf{refl}_a : a =_A a. Equivalence is A≃BA \simeq B. We use 𝒰\mathcal{U} for “the” universe. We deliberately distinguish three kinds of “equality”:

  • the judgmental / definitional equality (a meta-level notion), written a≡ba \equiv b, used in computation rules;

  • the propositional / identity-type equality, written a=ba = b, an inhabitant of the type 𝖨𝖽A(a,b)\mathsf{Id}_A(a, b);

  • type equivalence, written A≃BA \simeq B, an inhabitant of ∑f:A→B𝗂𝗌𝖤𝗊𝗎𝗂𝗏(f)\sum_{f : A \to B} \mathsf{isEquiv}(f).

Univalence asserts that the canonical map relating the second and third (at the universe level) is itself an equivalence.

Mathematical Framework

We assume Martin-Löf’s intensional dependent type theory MLTTω_\omega as the substrate . The pertinent rules are summarized below informally; for the formal calculus see .

Dependent products and sums

For a type family B:A→𝒰B : A \to \mathcal{U}, the Π\Pi-type ∏x:AB(x)\prod_{x : A} B(x) has introduction by λ\lambda-abstraction and elimination by application. The Σ\Sigma-type ∑x:AB(x)\sum_{x : A} B(x) has introduction (a,b)(a, b) for a:A,b:B(a)a : A, b : B(a) and elimination by projections 𝖿𝗌𝗍\mathsf{fst} and 𝗌𝗇𝖽\mathsf{snd} together with a dependent eliminator 𝗂𝗇𝖽Σ\mathsf{ind}_\Sigma.

Definition 1 (Function type). A→BA \to B abbreviates ∏x:AB\prod_{x : A} B when BB does not depend on xx.

Definition 2 (Cartesian product). A×BA \times B abbreviates ∑x:AB\sum_{x : A} B when BB does not depend on xx.

The unit, empty, and sum types

The unit type 𝟏\mathbf{1} has a single constructor ⋆:𝟏\star : \mathbf{1} and the eliminator “everything depending on 𝟏\mathbf{1} is determined by its value at ⋆\star.” The empty type 𝟎\mathbf{0} has no constructors and the eliminator 𝗂𝗇𝖽𝟎:∏C:𝟎→𝒰∏z:𝟎C(z)\mathsf{ind}_{\mathbf{0}} : \prod_{C : \mathbf{0} \to \mathcal{U}} \prod_{z : \mathbf{0}} C(z) (ex falso). The sum type A+BA + B has 𝗂𝗇𝗅:A→A+B\mathsf{inl}: A \to A + B and 𝗂𝗇𝗋:B→A+B\mathsf{inr}: B \to A + B.

Universes and Russell vs. Tarski

We adopt Russell-style universes: 𝒰0:𝒰1:⋯\mathcal{U}_0 : \mathcal{U}_1 : \cdots with 𝒰i⊂𝒰i+1\mathcal{U}_i \subset \mathcal{U}_{i+1} and a term of 𝒰\mathcal{U} is itself a type. All “𝒰\mathcal{U}”s in this paper denote a generic universe; statements are true at every level.

Natural Numbers as an Inductive Type

The inductive definition

Definition 3 (Natural numbers). The type ℕ:𝒰\mathbb{N}: \mathcal{U} is generated by the formation rule ℕ:𝒰\mathbb{N}: \mathcal{U} and the two constructors 𝗓𝖾𝗋𝗈:ℕ,𝗌𝗎𝖼𝖼:ℕ→ℕ.\begin{align*} \mathsf{zero}&: \mathbb{N}, \\ \mathsf{succ}&: \mathbb{N}\to \mathbb{N}. \end{align*} The dependent eliminator (induction principle) is 𝗂𝗇𝖽ℕ:∏P:ℕ→𝒰P(𝗓𝖾𝗋𝗈)→(∏n:ℕP(n)→P(𝗌𝗎𝖼𝖼n))→∏n:ℕP(n).\mathsf{ind}_\mathbb{N}: \prod_{P : \mathbb{N}\to \mathcal{U}}\; P(\mathsf{zero}) \to \Bigl(\prod_{n : \mathbb{N}} P(n) \to P(\mathsf{succ}\,n)\Bigr) \to \prod_{n : \mathbb{N}} P(n). The computation rules are 𝗂𝗇𝖽ℕPc0cs𝗓𝖾𝗋𝗈≡c0\mathsf{ind}_\mathbb{N}\,P\,c_0\,c_s\,\mathsf{zero}\equiv c_0 and 𝗂𝗇𝖽ℕPc0cs(𝗌𝗎𝖼𝖼n)≡csn(𝗂𝗇𝖽ℕPc0csn)\mathsf{ind}_\mathbb{N}\,P\,c_0\,c_s\,(\mathsf{succ}\,n) \equiv c_s\,n\,(\mathsf{ind}_\mathbb{N}\,P\,c_0\,c_s\,n).

Recursion and arithmetic

The non-dependent recursor 𝗋𝖾𝖼ℕ:∏C:𝒰C→(ℕ→C→C)→ℕ→C\mathsf{rec}_\mathbb{N}: \prod_{C : \mathcal{U}} C \to (\mathbb{N}\to C \to C) \to \mathbb{N}\to C is the special case where P(n):=CP(n) := C for all nn.

Example 5 (Addition). Define +:ℕ→ℕ→ℕ+ : \mathbb{N}\to \mathbb{N}\to \mathbb{N} by m+n:=𝗋𝖾𝖼ℕℕn(λk.𝗌𝗎𝖼𝖼)mm + n := \mathsf{rec}_\mathbb{N}\,\mathbb{N}\,n\,(\lambda k.\,\mathsf{succ})\,m, equivalently by 𝗓𝖾𝗋𝗈+n≡n\mathsf{zero}+ n \equiv n and (𝗌𝗎𝖼𝖼m)+n≡𝗌𝗎𝖼𝖼(m+n)(\mathsf{succ}\,m) + n \equiv \mathsf{succ}(m + n). Multiplication and exponentiation are defined analogously.

Proposition 6 (NNO universal property, internal). For any type C:𝒰C : \mathcal{U}, point c0:Cc_0 : C, and endo-map f:C→Cf : C \to C, there is a unique h:ℕ→Ch : \mathbb{N}\to C with h(𝗓𝖾𝗋𝗈)=c0h(\mathsf{zero}) = c_0 and h(𝗌𝗎𝖼𝖼n)=f(hn)h(\mathsf{succ}\,n) = f(h\,n) for all n:ℕn : \mathbb{N}.

Proof. Existence: h:=𝗋𝖾𝖼ℕCc0(λn.f)h := \mathsf{rec}_\mathbb{N}\,C\,c_0\,(\lambda n.\,f). Uniqueness: suppose h′:ℕ→Ch' : \mathbb{N}\to C also satisfies the equations. Define the predicate P(n):=(hn=Ch′n)P(n) := (h\,n =_C h'\,n).

Base case. The recursor’s computation rule gives the definitional (judgmental) equalities h(𝗓𝖾𝗋𝗈)≡c0h(\mathsf{zero}) \equiv c_0 and h′(𝗓𝖾𝗋𝗈)≡c0h'(\mathsf{zero}) \equiv c_0, so P(𝗓𝖾𝗋𝗈)P(\mathsf{zero}) is inhabited by 𝗋𝖾𝖿𝗅c0\mathsf{refl}_{c_0} – no propositional equality reasoning is needed.

Inductive step. Assume P(n)P(n), i.e. q:hn=h′nq : h\,n = h'\,n. The computation rules give h(𝗌𝗎𝖼𝖼n)≡f(hn)h(\mathsf{succ}\,n) \equiv f(h\,n) and h′(𝗌𝗎𝖼𝖼n)≡f(h′n)h'(\mathsf{succ}\,n) \equiv f(h'\,n) judgmentally; applying the action on paths 𝖺𝗉f:(a=b)→(fa=fb)\mathsf{ap}_f : (a = b) \to (f\,a = f\,b) yields 𝖺𝗉f(q):f(hn)=f(h′n)\mathsf{ap}_f(q) : f(h\,n) = f(h'\,n), i.e. h(𝗌𝗎𝖼𝖼n)=h′(𝗌𝗎𝖼𝖼n)h(\mathsf{succ}\,n) = h'(\mathsf{succ}\,n). Apply 𝗂𝗇𝖽ℕ\mathsf{ind}_\mathbb{N} to conclude ∏n:ℕP(n)\prod_{n : \mathbb{N}} P(n). Function extensionality (25) then gives h=h′h = h'. ◻

ℕ\mathbb{N} in HoTT vs. ℕ\mathbb{N} in set theory

Remark 7. Proposition 6 renders Benacerraf’s identification problem moot within the type theory: ℕ\mathbb{N} is determined up to canonical bijection by its formation and elimination rules. Combined with univalence (5) it is determined up to canonical equality as a term of 𝒰\mathcal{U}.

Identity Types and Path Induction

The identity type

Definition 8 (Identity type). For each type A:𝒰A : \mathcal{U} and elements a,b:Aa, b : A there is a type a=Aba =_A b, called the identity type or path space. It is the inductive family generated by the single constructor 𝗋𝖾𝖿𝗅a:a=Aa\mathsf{refl}_a : a =_A a for each a:Aa : A.

The eliminator – often called path induction or the JJ-rule – says that to prove a property of all paths it suffices to prove it of the constant paths.

Proposition 9 (Path induction, JJ). For every type A:𝒰A : \mathcal{U} and family C:∏x,y:A(x=Ay)→𝒰C : \prod_{x, y : A} (x =_A y) \to \mathcal{U}, given a section d:∏x:AC(x,x,𝗋𝖾𝖿𝗅x)d : \prod_{x : A} C(x, x, \mathsf{refl}_x), there is a section J(C,d):∏x,y:A∏p:x=AyC(x,y,p),J(C, d) : \prod_{x, y : A} \prod_{p : x =_A y} C(x, y, p), satisfying the computation rule J(C,d,x,x,𝗋𝖾𝖿𝗅x)≡d(x)J(C, d, x, x, \mathsf{refl}_x) \equiv d(x).

Remark 10. JJ is not the principle “every path is 𝗋𝖾𝖿𝗅\mathsf{refl}.” That stronger statement is Streicher’s KK-axiom and is incompatible with univalence: it forces every type to be a set. HoTT keeps JJ but rejects KK.

Symmetry, transitivity, and the path groupoid

Lemma 11 (Symmetry). There is a function 𝗌𝗒𝗆:∏A:𝒰∏x,y:A(x=Ay)→(y=Ax)\mathsf{sym} : \prod_{A : \mathcal{U}}\prod_{x, y : A} (x =_A y) \to (y =_A x).

Proof. Take C(x,y,p):=(y=Ax)C(x, y, p) := (y =_A x) and d(x):=𝗋𝖾𝖿𝗅xd(x) := \mathsf{refl}_x, then apply JJ. ◻

Lemma 12 (Transitivity). There is a function 𝗍𝗋𝖺𝗇𝗌:∏A:𝒰∏x,y,z:A(x=Ay)→(y=Az)→(x=Az)\mathsf{trans} : \prod_{A : \mathcal{U}}\prod_{x, y, z : A} (x =_A y) \to (y =_A z) \to (x =_A z).

Proof. Fix A,x,y,zA, x, y, z and p:x=yp : x = y. By JJ on pp it suffices to assume y≡xy \equiv x and p≡𝗋𝖾𝖿𝗅xp \equiv \mathsf{refl}_x, in which case 𝗍𝗋𝖺𝗇𝗌(𝗋𝖾𝖿𝗅x,q):=q\mathsf{trans}(\mathsf{refl}_x, q) := q is a section. ◻

We write p−1p^{-1} for 𝗌𝗒𝗆(p)\mathsf{sym}(p) and p⋅qp \cdot q for 𝗍𝗋𝖺𝗇𝗌(p,q)\mathsf{trans}(p, q).

Theorem 13 (Path groupoid). For any type AA and a,b,c,d:Aa, b, c, d : A, p:a=bp : a = b, q:b=cq : b = c, r:c=dr : c = d:

  1. 𝗋𝖾𝖿𝗅a⋅p=p\mathsf{refl}_a \cdot p = p and p⋅𝗋𝖾𝖿𝗅b=pp \cdot \mathsf{refl}_b = p;

  2. p⋅p−1=𝗋𝖾𝖿𝗅ap \cdot p^{-1} = \mathsf{refl}_a and p−1⋅p=𝗋𝖾𝖿𝗅bp^{-1} \cdot p = \mathsf{refl}_b;

  3. (p⋅q)⋅r=p⋅(q⋅r)(p \cdot q) \cdot r = p \cdot (q \cdot r).

Moreover the higher coherences (the pentagon, the Eckmann–Hilton argument, etc.) all hold up to higher paths, making AA into an ∞\infty-groupoid .

Proof sketch. Each clause is proved by JJ on the relevant paths. For the higher coherences one iterates JJ at successively higher levels; see . ◻

Transport

Definition 14 (Transport). For P:A→𝒰P : A \to \mathcal{U}, a,b:Aa, b : A, and p:a=bp : a = b, the function 𝗍𝗋𝖺𝗇𝗌𝗉𝗈𝗋𝗍P(p):P(a)→P(b)\mathsf{transport}^P(p) : P(a) \to P(b) is defined by JJ: 𝗍𝗋𝖺𝗇𝗌𝗉𝗈𝗋𝗍P(𝗋𝖾𝖿𝗅a)(u):=u\mathsf{transport}^P(\mathsf{refl}_a)(u) := u.

Proposition 15 (Transport along concatenation). 𝗍𝗋𝖺𝗇𝗌𝗉𝗈𝗋𝗍P(p⋅q)=𝗍𝗋𝖺𝗇𝗌𝗉𝗈𝗋𝗍P(q)∘𝗍𝗋𝖺𝗇𝗌𝗉𝗈𝗋𝗍P(p)\mathsf{transport}^P(p \cdot q) = \mathsf{transport}^P(q) \circ \mathsf{transport}^P(p).

Proof. JJ on either pp or qq. ◻

Homotopy levels

Definition 16 (Contractibility, propositions, sets). A type A:𝒰A : \mathcal{U} is

  • contractible, 𝗂𝗌𝖢𝗈𝗇𝗍𝗋(A):=∑c:A∏x:A(x=c)\mathsf{isContr}(A) := \sum_{c : A} \prod_{x : A} (x = c);

  • a mere proposition, 𝗂𝗌𝖯𝗋𝗈𝗉(A):=∏x,y:A(x=y)\mathsf{isProp}(A) := \prod_{x, y : A} (x = y);

  • a set (h-set), 𝗂𝗌𝖲𝖾𝗍(A):=∏x,y:A𝗂𝗌𝖯𝗋𝗈𝗉(x=y)\mathsf{isSet}(A) := \prod_{x, y : A} \mathsf{isProp}(x = y).

The hierarchy continues: AA is an nn-type iff every identity type of AA is an (n−1)(n-1)-type.

Theorem 17 (Hedberg). If AA has decidable equality ∏x,y:A(x=y)+¬(x=y)\prod_{x, y : A} (x = y) + \neg(x = y), then AA is a set.

Corollary 18. ℕ\mathbb{N} is a set.

Proof. Equality on ℕ\mathbb{N} is decidable by induction on both arguments and Peano’s fourth axiom 𝗌𝗎𝖼𝖼m=𝗌𝗎𝖼𝖼n⇒m=n\mathsf{succ}\,m = \mathsf{succ}\,n \Rightarrow m = n. Apply Hedberg. ◻

The Univalence Axiom

Equivalence of types

Definition 19 (Quasi-inverse and equivalence). A function f:A→Bf : A \to B has a quasi-inverse if there exist g:B→Ag : B \to A and homotopies η:g∘f∼idA\eta : g \circ f \sim \mathrm{id}_A and ε:f∘g∼idB\varepsilon : f \circ g \sim \mathrm{id}_B. The proposition 𝗂𝗌𝖤𝗊𝗎𝗂𝗏(f)\mathsf{isEquiv}(f) is the half-adjoint version , ensuring 𝗂𝗌𝖤𝗊𝗎𝗂𝗏(f)\mathsf{isEquiv}(f) is itself a mere proposition. We write A≃B:=∑f:A→B𝗂𝗌𝖤𝗊𝗎𝗂𝗏(f)A \simeq B := \sum_{f : A \to B} \mathsf{isEquiv}(f).

Proposition 20 (idtoeqv). There is a canonical map 𝗂𝖽𝗍𝗈𝖾𝗊𝗏:(A=𝒰B)→(A≃B)\mathsf{idtoeqv}: (A =_\mathcal{U}B) \to (A \simeq B) taking 𝗋𝖾𝖿𝗅A\mathsf{refl}_A to the identity equivalence.

Proof. JJ on the path A=BA = B. ◻

The axiom

Theorem 21 (Voevodsky’s univalence axiom). (; .) For all types A,B:𝒰A, B : \mathcal{U}, the canonical map 𝗂𝖽𝗍𝗈𝖾𝗊𝗏\mathsf{idtoeqv} is itself an equivalence: 𝗎𝖺:(A≃B)→(A=𝒰B),𝗂𝖽𝗍𝗈𝖾𝗊𝗏∘𝗎𝖺=idA≃B,𝗎𝖺∘𝗂𝖽𝗍𝗈𝖾𝗊𝗏=idA=𝒰B.\mathsf{ua}: (A \simeq B) \to (A =_\mathcal{U}B), \qquad \mathsf{idtoeqv}\circ \mathsf{ua}= \mathrm{id}_{A \simeq B}, \qquad \mathsf{ua}\circ \mathsf{idtoeqv}= \mathrm{id}_{A =_\mathcal{U}B}.

In Book HoTT this is taken as an axiom; in cubical type theory (7) it is a theorem.

Consequences for arithmetic

Definition 22 (Type of NNOs). Define NNO:=∑X:𝒰∑x0:X∑s:X→X𝗂𝗌𝖨𝗇𝗂𝗍𝗂𝖺𝗅(X,x0,s),\mathrm{NNO} := \sum_{X : \mathcal{U}}\;\sum_{x_0 : X}\;\sum_{s : X \to X}\; \mathsf{isInitial}(X, x_0, s), where 𝗂𝗌𝖨𝗇𝗂𝗍𝗂𝖺𝗅(X,x0,s)\mathsf{isInitial}(X, x_0, s) is the universal property of 6 expressed as a Σ\Sigma-type.

Theorem 23 (Contractibility of NNO). NNO\mathrm{NNO} is contractible.

Proof. We exhibit (ℕ,𝗓𝖾𝗋𝗈,𝗌𝗎𝖼𝖼,𝗂𝗇𝗂𝗍ℕ)(\mathbb{N}, \mathsf{zero}, \mathsf{succ}, \mathsf{init}_\mathbb{N}) as the centre and supply a proof that every other quadruple is equal to it.

Step 1. Equivalence at the carrier. By 6 the standard structure (ℕ,𝗓𝖾𝗋𝗈,𝗌𝗎𝖼𝖼)(\mathbb{N}, \mathsf{zero}, \mathsf{succ}) is initial. For any other initial (X,x0,s)(X, x_0, s), the universal property applied in both directions yields mutually inverse maps f:ℕ→Xf : \mathbb{N}\to X and g:X→ℕg : X \to \mathbb{N} with g∘f=idℕg \circ f = \mathrm{id}_\mathbb{N} and f∘g=idXf \circ g = \mathrm{id}_X (each by uniqueness of the universal recursor). Half-adjointness packages this as an equivalence e:ℕ≃Xe : \mathbb{N}\simeq X. Univalence (21) promotes this to a path p:=𝗎𝖺(e):ℕ=𝒰Xp := \mathsf{ua}(e) : \mathbb{N}=_\mathcal{U}X.

Step 2. Paths in Σ\Sigma-types. Recall that a path in ∑a:AB(a)\sum_{a : A} B(a) between (a,b)(a, b) and (a′,b′)(a', b') is, up to canonical isomorphism, a pair (p,q)(p, q) where p:a=Aa′p : a =_A a' and q:𝗍𝗋𝖺𝗇𝗌𝗉𝗈𝗋𝗍B(p)(b)=B(a′)b′q : \mathsf{transport}^B(p)(b) =_{B(a')} b' . Iterating: a path between two iterated Σ\Sigma-tuples (a1,a2,…,an)(a_1, a_2, \ldots, a_n) and (a1′,a2′,…,an′)(a_1', a_2', \ldots, a_n') is a tuple of paths (p1,p2,…,pn)(p_1, p_2, \ldots, p_n) where pkp_k lives over the preceding p1,…,pk−1p_1, \ldots, p_{k-1}.

Step 3. Constructing the path componentwise. We must produce paths (p,p0,ps,p*)(p, p_0, p_s, p_*) where:

  1. p:ℕ=𝒰Xp : \mathbb{N}=_\mathcal{U}X is the carrier path from Step 1.

  2. p0:𝗍𝗋𝖺𝗇𝗌𝗉𝗈𝗋𝗍(X↦X)(p)(𝗓𝖾𝗋𝗈)=Xx0p_0 : \mathsf{transport}^{(X \mapsto X)}(p)(\mathsf{zero}) =_X x_0. Because 𝗎𝖺(e)\mathsf{ua}(e) transports along the equivalence ee, the left-hand side reduces to e(𝗓𝖾𝗋𝗈)e(\mathsf{zero}), and the universal property of ℕ\mathbb{N} forces e(𝗓𝖾𝗋𝗈)=x0e(\mathsf{zero}) = x_0 (the unique map ℕ→X\mathbb{N}\to X sends 𝗓𝖾𝗋𝗈\mathsf{zero} to x0x_0), giving the required path.

  3. ps:𝗍𝗋𝖺𝗇𝗌𝗉𝗈𝗋𝗍(X↦X→X)(p)(𝗌𝗎𝖼𝖼)=sp_s : \mathsf{transport}^{(X \mapsto X \to X)}(p)(\mathsf{succ}) = s. The type family X↦X→XX \mapsto X \to X has transport given by conjugation by ee, so the left-hand side reduces to e∘𝗌𝗎𝖼𝖼∘e−1e \circ \mathsf{succ}\circ e^{-1}. Naturality of the universal recursor yields e∘𝗌𝗎𝖼𝖼=s∘ee \circ \mathsf{succ}= s \circ e, whence e∘𝗌𝗎𝖼𝖼∘e−1=se \circ \mathsf{succ}\circ e^{-1} = s, supplying psp_s.

  4. p*p_*: the initiality witness 𝗂𝗌𝖨𝗇𝗂𝗍𝗂𝖺𝗅\mathsf{isInitial} is a mere proposition (uniqueness of the universal recursor is a Pi-type into a contractible identity type), so any two initiality proofs are identified by a unique path.

Step 4. Conclusion. Bundling (p,p0,ps,p*)(p, p_0, p_s, p_*) via the iterated Σ\Sigma-path constructor gives the required equality (ℕ,𝗓𝖾𝗋𝗈,𝗌𝗎𝖼𝖼,𝗂𝗇𝗂𝗍ℕ)=NNO(X,x0,s,⋅)(\mathbb{N}, \mathsf{zero}, \mathsf{succ}, \mathsf{init}_\mathbb{N}) =_{\mathrm{NNO}} (X, x_0, s, \cdot). Since (X,x0,s,⋅)(X, x_0, s, \cdot) was arbitrary, NNO\mathrm{NNO} is contractible. The argument is the same shape as the contractibility of the type of “small structures-of-shape-TT on a fixed base” once TT is replaced by the signature “one constant and one unary operation”; see . ◻

Corollary 24. For any NNO (X,x0,s)(X, x_0, s) the term s58x0s^{58}\,x_0 is, after transport along the canonical equality ℕ=X\mathbb{N}= X, equal to 58¯:ℕ\overline{58} : \mathbb{N}.

Higher Inductive Types

A higher inductive type (HIT) extends inductive types with constructors for paths as well as points.

The interval and propositional truncation

Definition 26 (Interval). The type II has constructors 0I,1I:I0_I, 1_I : I and a path 𝗌𝖾𝗀:0I=1I\mathsf{seg} : 0_I = 1_I.

Definition 27 (Propositional truncation). ∥A∥\|A\| has |a|:∥A∥|a| : \|A\| for a:Aa : A and a path constructor ∏x,y:∥A∥(x=y)\prod_{x, y : \|A\|}(x = y). Its eliminator allows mapping ∥A∥→P\|A\| \to P when PP is a mere proposition.

The circle

Definition 28 (Circle). S1S^1 has constructors 𝖻𝖺𝗌𝖾:S1\mathsf{base}: S^1 and a path 𝗅𝗈𝗈𝗉:𝖻𝖺𝗌𝖾=S1𝖻𝖺𝗌𝖾\mathsf{loop}: \mathsf{base}=_{S^1} \mathsf{base}. The eliminator: given P:S1→𝒰P : S^1\to \mathcal{U}, b:P(𝖻𝖺𝗌𝖾)b : P(\mathsf{base}), and ℓ:𝗍𝗋𝖺𝗇𝗌𝗉𝗈𝗋𝗍P(𝗅𝗈𝗈𝗉)(b)=b\ell : \mathsf{transport}^P(\mathsf{loop})(b) = b, there is 𝗂𝗇𝖽S1(P,b,ℓ):∏x:S1P(x)\mathsf{ind}_{S^1}(P, b, \ell) : \prod_{x : S^1} P(x), computing as 𝗂𝗇𝖽S1(P,b,ℓ)(𝖻𝖺𝗌𝖾)≡b\mathsf{ind}_{S^1}(P, b, \ell)(\mathsf{base}) \equiv b and a corresponding rule on 𝗅𝗈𝗈𝗉\mathsf{loop}.

Theorem 29 (Loop space of S1S^1{}). π1(S1):=∥(𝖻𝖺𝗌𝖾=𝖻𝖺𝗌𝖾)∥0≃ℤ\pi_1(S^1) := \|(\mathsf{base}= \mathsf{base})\|_0 \simeq\mathbb{Z}. Indeed (𝖻𝖺𝗌𝖾=𝖻𝖺𝗌𝖾)≃ℤ(\mathsf{base}= \mathsf{base}) \simeq\mathbb{Z} as a type, before truncation.

Proof sketch. Define the universal cover 𝖼𝗈𝖽𝖾:S1→𝒰\mathsf{code} : S^1\to \mathcal{U} by 𝖼𝗈𝖽𝖾(𝖻𝖺𝗌𝖾):=ℤ\mathsf{code}(\mathsf{base}) := \mathbb{Z} and on 𝗅𝗈𝗈𝗉\mathsf{loop} by univalence and the successor equivalence 𝗌𝗎𝖼𝖼:ℤ≃ℤ\mathsf{succ} : \mathbb{Z}\simeq\mathbb{Z}. Then (𝖻𝖺𝗌𝖾=−)∼𝖼𝗈𝖽𝖾(\mathsf{base}= -) \sim \mathsf{code} by the encode–decode method (Licata–Shulman ); evaluating at 𝖻𝖺𝗌𝖾\mathsf{base} gives (𝖻𝖺𝗌𝖾=𝖻𝖺𝗌𝖾)≃ℤ(\mathsf{base}= \mathsf{base}) \simeq\mathbb{Z}. ◻

29 is the first non-trivial theorem of synthetic homotopy theory: a result of algebraic topology proved entirely inside type theory, without recourse to topological spaces, simplicial sets, or model categories.

Spheres, suspensions, and pushouts

The nn-sphere SnS^n is defined recursively as ΣSn−1\Sigma S^{n-1} where Σ\Sigma is the unreduced suspension HIT. The pushout of f:C→Af : C \to A and g:C→Bg : C \to B has constructors for AA, BB, and a path identifying f(c)f(c) and g(c)g(c) for each c:Cc : C.

Theorem 30 (Hopf fibration, internal). There is a fibration h:S3→S2h : S^3 \to S^2 with fibers S1S^1{}, definable as a HIT and characterized by the long exact sequence of homotopy groups; in particular π3(S2)≃ℤ\pi_3(S^2) \simeq\mathbb{Z} via the Hopf invariant .

Eilenberg–MacLane spaces

For each abelian group GG and n≥1n \geq 1 there is a HIT K(G,n)K(G, n) with πn=G\pi_n = G and πk=0\pi_k = 0 for k≠nk \neq n . Synthetic cohomology Hn(X;G):=∥X→K(G,n)∥0H^n(X; G) := \|X \to K(G, n)\|_0 recovers ordinary singular cohomology when XX is a CW-complex.

Real numbers as a higher inductive–inductive type

Theorem 31 (Booij; HoTT Book §11.3). (, after .) There is a higher inductive–inductive type ℝc\mathbb{R}_c presenting the Cauchy reals: an h-set with constructors for rational injections 𝗋𝖺𝗍:ℚ→ℝc\mathsf{rat} : \mathbb{Q}\to \mathbb{R}_c, a Cauchy-completion limit operation, and a path constructor identifying terms whose distance witnesses are below every positive rational. ℝc\mathbb{R}_c has the universal property of the Cauchy completion of ℚ\mathbb{Q} in the category of metric spaces internal to HoTT, and satisfies the axioms of an Archimedean ordered field.

The construction is higher because of the path constructor and inductive–inductive because the path constructor mentions a “closeness” relation defined simultaneously with the carrier type.

Worked example: the canonical 58 across NNOs

To make 23 concrete, we walk through how an arbitrary NNO yields the same “58” as ℕ\mathbb{N}.

Example 32 (58 in two NNOs). Take two structures.

  • The standard one: (ℕ,𝗓𝖾𝗋𝗈,𝗌𝗎𝖼𝖼)(\mathbb{N}, \mathsf{zero}, \mathsf{succ}).

  • A bit-string encoding: X:=List(𝟏)X := \mathrm{List}(\mathbf{1}), with x0:=𝗇𝗂𝗅x_0 := \mathsf{nil} and s:=𝖼𝗈𝗇𝗌(⋆,−)s := \mathsf{cons}(\star, -).

Both are easily checked to satisfy the universal property of 6; for the bit-string version, the unique recursor sends a list of length nn to the value fn(c0)f^n(c_0) in any target (C,c0,f)(C, c_0, f).

By 23, the data (ℕ,𝗓𝖾𝗋𝗈,𝗌𝗎𝖼𝖼)(\mathbb{N}, \mathsf{zero}, \mathsf{succ}) and (X,x0,s)(X, x_0, s) are equal as inhabitants of NNO\mathrm{NNO}. Concretely:

  1. the equivalence e:ℕ≃Xe : \mathbb{N}\simeq X sends n¯\overline{n} to the list of length nn;

  2. univalence gives 𝗎𝖺(e):ℕ=𝒰X\mathsf{ua}(e) : \mathbb{N}=_\mathcal{U}X;

  3. transporting 58¯:ℕ\overline{58} : \mathbb{N} along 𝗎𝖺(e)\mathsf{ua}(e) yields the 58-element list of ⋆\star’s in XX;

  4. this list is, by definition of XX, the value of s58x0s^{58}\,x_0.

Thus the two “58”s are propositionally equal, and this very equality is the content of 24.

Remark 33 (The Yoneda perspective recovered). Paper 04 characterises every object AA by its representable functor Hom(−,A)\mathrm{Hom}(-, A) and shows A≃B⇔Hom(−,A)≃Hom(−,B)A \simeq B \Leftrightarrow \mathrm{Hom}(-, A) \simeq \mathrm{Hom}(-, B) (Yoneda). Univalence upgrades the right-to-left direction: equivalent representables yield equal objects in 𝒰\mathcal{U}, which is the type-theoretic statement of the structuralist slogan “a thing is what it relates to.”

Worked example: the loop concatenation as integer addition

For 29, let us trace what happens to specific loops.

Example 34 (Three full turns then two backward). Inside HoTT we represent the loop “three times around” as the path 𝗅𝗈𝗈𝗉⋅𝗅𝗈𝗈𝗉⋅𝗅𝗈𝗈𝗉\mathsf{loop}\cdot \mathsf{loop}\cdot \mathsf{loop} at 𝖻𝖺𝗌𝖾\mathsf{base} in S1S^1, i.e. a term of 𝖻𝖺𝗌𝖾=𝖻𝖺𝗌𝖾\mathsf{base}= \mathsf{base}. By the encode–decode equivalence (𝖻𝖺𝗌𝖾=𝖻𝖺𝗌𝖾)≃ℤ(\mathsf{base}= \mathsf{base}) \simeq\mathbb{Z}, this term encodes to the integer 33.

Composing with 𝗅𝗈𝗈𝗉−1⋅𝗅𝗈𝗈𝗉−1\mathsf{loop}^{-1} \cdot \mathsf{loop}^{-1} gives, by 13, the path 𝗅𝗈𝗈𝗉⋅𝗅𝗈𝗈𝗉⋅𝗅𝗈𝗈𝗉⋅𝗅𝗈𝗈𝗉−1⋅𝗅𝗈𝗈𝗉−1\mathsf{loop}\cdot \mathsf{loop}\cdot \mathsf{loop}\cdot \mathsf{loop}^{-1} \cdot \mathsf{loop}^{-1}, which the encode-decode equivalence sends to 3+(−1)+(−1)=13 + (-1) + (-1) = 1.

Decoding back yields a single 𝗅𝗈𝗈𝗉\mathsf{loop}, propositionally equal to the original concatenation. The point is not that we computed 3−2=13 - 2 = 1 (routine!), but that this very arithmetic of ℤ\mathbb{Z} is the fundamental group law on π1(S1)\pi_1(S^1) once we identify the two via the equivalence.

Cubical Type Theory

The original presentation of HoTT takes univalence as an axiom, breaking canonicity: a closed term of ℕ\mathbb{N} may be propositionally but not definitionally equal to a numeral. Cubical type theory restores canonicity.

Intervals and faces

Definition 35 (Cube category). Cohen–Coquand–Huber–Mörtberg  introduce a primitive interval object 𝕀\mathbb{I} generated by 0,1:𝕀0, 1 : \mathbb{I}, the operations ⊔,⊓,¬\sqcup, \sqcap, \neg making 𝕀\mathbb{I} into a de Morgan algebra, and the algebra of face formulas φ\varphi over the generating equations i=0i = 0, i=1i = 1.

A path p:a=Abp : a =_A b is then a function 𝕀→A\mathbb{I} \to A with p(0)=ap(0) = a and p(1)=bp(1) = b judgmentally. Path concatenation, inversion, and dependent elimination become explicit term operations on cubical sets, not derived from JJ.

Composition and Glue

The crucial operations are composition (filling open boxes) and Glue (gluing types along an equivalence). The Glue type 𝖦𝗅𝗎𝖾A[φ↦(T,f)]\mathsf{Glue}\,A\,[\varphi \mapsto (T, f)] where f:T≃Af : T \simeq A on the face φ\varphi supplies a constructive proof of univalence: every equivalence is realized as a path in 𝒰\mathcal{U}.

Theorem 36 (Univalence in cubical type theory). Cubical type theory in the CCHM model proves 𝗎𝖺\mathsf{ua} as a defined term, and 𝗂𝖽𝗍𝗈𝖾𝗊𝗏∘𝗎𝖺\mathsf{idtoeqv}\circ \mathsf{ua} reduces to the identity by computation, not by postulate .

Implementations

The Cubical Agda mode  and the cubicaltt prototype implement the system; the redtt and cooltt projects extend it with cartesian intervals and modalities. For our purposes the relevant fact is that all theorems of [sec:univalence,sec:hits] can be computed, not just postulated.

Variants: 2LTT, Modal, and Directed HoTT

Two-level type theory

Two-level type theory (Annenkov–Capriotti–Kraus–Sattler ) introduces a strict equality alongside the homotopical one. The strict fragment is used to formalize semantics (e.g. inverse diagrams of types representing semisimplicial types) that the homotopical fragment alone cannot encode without higher coherence. Strict equality is incompatible with univalence, so 2LTT carefully separates the two universes.

Modal HoTT

Modal type theory adds operators ▫,♢\square, \diamondsuit corresponding to adjunctions in the categorical semantics. Examples include Schreiber–Shulman cohesive HoTT , where the cohesion modalities recover the discrete–codiscrete–shape adjoint string disc⊣Π⊣codisc\mathrm{disc} \dashv \Pi \dashv \mathrm{codisc} for cohesive (∞,1)(\infty,1)-toposes, and the spatial type theory of Riley .

Directed HoTT

Directed type theory replaces invertible paths with possibly non-invertible morphisms, so that types correspond to (∞,n)(\infty, n)-categories rather than ∞\infty-groupoids. Riehl–Shulman  develop simplicial type theory with a primitive directed interval Δ1\Delta^1 and prove a synthetic Yoneda lemma; directed univalence is an active research frontier.

Transcendentals in HoTT

π\pi as the period of exp\exp on iℝi\mathbb{R}

Definition 37 (Real exponential). The map exp⁡:ℝc→ℝc\exp : \mathbb{R}_c \to \mathbb{R}_c is defined as the unique solution to y′=y,y(0)=1y' = y, y(0) = 1 in ℝc\mathbb{R}_c – equivalently, by the power series ∑n=0∞xn/n!\sum_{n=0}^{\infty} x^n / n!, which converges in ℝc\mathbb{R}_c by completeness.

Theorem 39. Lindemann’s theorem ports to HoTT: π\pi is transcendental over ℚ\mathbb{Q}, in the sense that the type ∑p:Poly(ℚ)(p≠0)×(p(π)=0)\sum_{p : \mathrm{Poly}(\mathbb{Q})}\;\bigl(p \neq 0\bigr) \times \bigl(p(\pi) = 0\bigr) is empty (equivalent to 𝟎\mathbf{0}).

Proof sketch. Lindemann’s classical proof relies only on convergent power series, elementary symmetric functions, and bounding arguments – all of which are available constructively in ℝc\mathbb{R}_c. The port is therefore believed to be possible based on the constructive character of the classical argument; however, to the authors’ knowledge no end-to-end formalization in HoTT is yet complete. The presently available scaffolding includes:

  • Cauchy completeness of ℝc\mathbb{R}_c, basic limits, the Archimedean property, and elementary calculus on ℝc\mathbb{R}_c (Booij’s thesis , expanding ).

  • Convergence of power series and analytic functions in HoTT, plus algebraic manipulations of polynomial rings and symmetric functions (scaffolded in the agda-unimath project ).

The pieces still missing for an end-to-end formalization are: (i) the Hermite–Lindemann auxiliary function J(t)=∫0tet−sf(s)dsJ(t) = \int_0^t e^{t - s} f(s)\,ds and its bounds in ℝc\mathbb{R}_c; (ii) the algebraic-integer estimates that drive the contradiction; and (iii) the integration with a finished formalization of the Cauchy integral over ℝc\mathbb{R}_c. We identify this as open work in 15. ◻

Identification across encodings

Theorem 42 (Reals are unique in HoTT up to equivalence). Any two h-sets satisfying the universal property of the Cauchy reals (or the Dedekind reals, modulo the law of excluded middle for the latter) are equivalent, hence equal as types by univalence. Consequently, π\pi and ee are well-defined independently of the chosen construction.

Bridge: HoTT as the Internal Language of (∞,1)(\infty,1)-Toposes

Categorical semantics

Theorem 43 (Awodey–Warren). (.) Identity types of intensional Martin-Löf type theory are interpreted in weak factorisation systems; the model in simplicial sets exhibits the path groupoid as a Kan complex.

Voevodsky’s simplicial set model  establishes that HoTT with univalence has a sound interpretation in the (∞,1)(\infty, 1)-topos 𝗌𝖲𝖾𝗍\mathsf{sSet} of Kan complexes.

Theorem 44 (Shulman). (.) Every (∞,1)(\infty, 1)-topos ℰ\mathcal{E} admits a model of HoTT with univalence; conversely, the syntactic category of HoTT is conjecturally the initial (∞,1)(\infty, 1)-topos.

Co-extensiveness with categorical and structural perspectives

The Yoneda embedding (Paper 04) and the categorical structuralism of Paper 06 have always been “equivalent up to equivalence.” Univalence makes them equal – equal as objects of 𝒰\mathcal{U}, equal as theories under semantic interpretation, equal as derivations modulo a translation that is itself an equivalence.

Remark 45 (The univalent correspondence, formalized). The Curry–Howard–Lambek tradition (see Wadler’s exposition ) identifies logic, type theory, and category theory as a triple. The four-column correspondence Logic↔Type Theory↔Category Theory↔Homotopy\text{Logic} \;\longleftrightarrow\; \text{Type Theory} \;\longleftrightarrow\; \text{Category Theory} \;\longleftrightarrow\; \text{Homotopy} is, in HoTT, the statement that the four interpretations of the same term in 𝒰\mathcal{U} correspond to the same homotopy type, with the categorical column interpreted in (∞,1)(\infty, 1)-toposes (Lurie’s higher topos theory  provides the ambient setting). In particular, “58 as a proposition” (the inhabited type 𝖥𝗂𝗇59∩𝗉𝗋𝗂𝗆𝖾c\mathsf{Fin}_{59} \cap \mathsf{prime}^c, etc.), “58 as a term” (58¯:ℕ\overline{58} : \mathbb{N}), “58 as an object” (the discrete object 58∈𝖲𝖾𝗍58 \in \mathsf{Set}), and “58 as a space” (the discrete homotopy type 𝖡𝟏58\mathsf{B}\mathbf{1}^{58}) are equal up to canonical paths.

Results

We collect the principal contributions of this paper.

Theorem 46 (Encoding-free arithmetic). In Book HoTT with univalence, the following are equivalent:

  1. the inductive type ℕ\mathbb{N} with 𝗓𝖾𝗋𝗈,𝗌𝗎𝖼𝖼\mathsf{zero}, \mathsf{succ};

  2. the initial NNO in any model of HoTT with ℕ\mathbb{N}-types;

  3. the unique pointed endo-structure XX with 𝗂𝗌𝖨𝗇𝗂𝗍𝗂𝖺𝗅(X,x0,s)\mathsf{isInitial}(X, x_0, s);

  4. up to univalent equality, every other type satisfying any of (1)–(3).

The witnesses to these equivalences are themselves canonical (paths in 𝒰\mathcal{U}), so 58:=58¯58 := \overline{58} is independent of representation.

Theorem 47 (Synthetic computation of π1(S1)\pi_1(S^1{})). Within HoTT, π1(S1)=ℤ\pi_1(S^1{}) = \mathbb{Z} admits a fully formal proof that mechanizes in Agda and Cubical Agda; the proof uses no external topology, only the HIT S1S^1 and univalence.

Theorem 48 (Reals and transcendentals). The Cauchy reals ℝc\mathbb{R}_c form an h-set inside HoTT (31). The terms π\pi and ee are unique-existence centres of contractible types of solutions of structural conditions. Their classical properties (irrationality, transcendence) port mechanically.

Theorem 49 (Bridge theorem). HoTT is sound and complete for (∞,1)(\infty, 1)-topos semantics . Hence the categorical (Paper 06), structural (Paper 07), Yoneda (Paper 04), and HoTT (Paper 05) views of arithmetic are co-extensive: a statement provable in any one is provable in any other, modulo translation.

Discussion

What HoTT does not solve

HoTT is a foundation, not a panacea. The following remain open:

  • Coherence beyond truncation level: proving general coherence theorems for (∞,1)(\infty, 1)-categories inside HoTT (rather than via 2LTT) is non-trivial. The semisimplicial types problem is the canonical example.

  • Decidability and computation under axioms: Book HoTT loses canonicity. Cubical type theory restores it but at the cost of a more intricate calculus.

  • Set-theoretic strength: HoTT with univalence has the proof-theoretic strength of ZFC + “there exist many inaccessible cardinals” , comparable to standard mathematical practice. Whether all of analytic number theory ports remains an open research front (Paper 07 surveys this).

Philosophical reading

The shift from set theory to HoTT mirrors the shift from substantialism to structuralism. “58 is a thing” becomes “58 is a position in a structure” becomes “58 is the equivalence class of all positions in isomorphic structures” becomes (in HoTT) “58 is a term whose identity is witnessed by paths.” Univalence collapses the apparent distinction between the last two.

Pedagogical implications

Synthetic homotopy theory provides a path-from-paths approach to algebraic topology: π1(S1)=ℤ\pi_1(S^1{}) = \mathbb{Z} in twelve lines of Agda. This is not merely a streamlining; it indicates that homotopy is more primitive than topology in the sense that the latter can be dispensed with.

Computational realisation

The Haskell companion (13) and the Agda fragments (14) demonstrate that path induction, transport, and the basic equational reasoning calculus can be simulated at the type level in mainstream functional languages, even without full dependent types. This is a useful pedagogical bridge, not a replacement.

Comparison with set-theoretic and categorical foundations

It is illuminating to put the four foundational stances side by side, as they answer the question “what is the natural number 5858?”.

  • Set theory (Paper 02). “5858 is the von Neumann ordinal {0,1,…,57}\{0, 1, \ldots, 57\}.” The encoding is canonical only by stipulation; Benacerraf’s identification problem records the residual arbitrariness.

  • Categorical foundations (Paper 03 / Paper 04). “5858 is the value of the unique recursion morphism on the natural-number object, equivalently a position in the representable Hom(−,ℕ)\mathrm{Hom}(-, \mathbb{N}).” Encoding-independence holds up to canonical isomorphism; the question “but which NNO?” remains a residue, albeit a small one.

  • HoTT (this paper). “5858 is 58¯:ℕ\overline{58} : \mathbb{N}, where ℕ\mathbb{N} is determined up to a canonical equality of types by univalence.” Encoding-independence is internal: there is no extra- theoretic notion of “which ℕ\mathbb{N}.”

  • Structural / ∞\infty-topos perspective (Paper 06). “5858 is an invariant under all structure-preserving maps,” which by the internal-language theorem is the same as the HoTT statement.

The progression is one of decreasing residue: each step removes a notion that the previous level could not eliminate.

When axiomatic univalence cannot be avoided

In Book HoTT, univalence is not a constructive principle: a closed term of ℕ\mathbb{N} produced via univalence may fail to reduce to a numeral. For many proofs this matters little (we only care about propositional equality), but for executable extraction it is fatal. Cubical type theory solves this; some applications – notably synthetic differential geometry – additionally require modal axioms that themselves are not yet known to be cubically computational. The trade-off between axiomatic generality and definitional reduction is an active research front.

The most natural ambient language for HoTT is Agda or Cubical Agda. We include here an Agda-flavoured sketch of the basic constructions; full files for Cubical Agda live alongside the Haskell sources.

Conclusion and Future Work

We have argued that Homotopy Type Theory provides the synthetic language in which arithmetic, analysis, and algebraic topology share a single foundational system. The natural number 5858 is the canonical 58¯:ℕ\overline{58} : \mathbb{N}; the universe-level identification of all NNOs is a theorem; the transcendental constants π\pi and ee are unique-existence centres in the higher inductive–inductive type ℝc\mathbb{R}_c. Univalence makes the categorical and structural perspectives co-extensive with the type-theoretic one.

The companion Haskell code provides a hands-on, executable mirror of the key constructions. The next paper (Paper 06) develops the categorical / structural perspective independently and shows that it merges with the HoTT perspective via the internal language theorem; Paper 07 (the synthesis) closes the loop.

Open problems

  1. Formalize Lindemann–Weierstrass in Cubical Agda end-to-end.

  2. Develop directed univalence sufficient to express the synthetic (∞,2)(\infty, 2)-Yoneda lemma.

  3. Formalize the Brown representability theorem inside synthetic homotopy theory; this would clarify the relation to the spectra constructions in classical algebraic topology.
